# Chatter.ai

## What is chatter.ai?
It is a way to incorporate Ai through text messages. It also serves as a web application to talk to AI

## Web-App
* Front-End: React
* Back-End: Python
* APIs: ChatGPT, Stable Diffusion, Twilio

## How to Start
-Run front-end
  npm start

-Run server
  cd server
  .venv\Scripts\activate
  flask run
